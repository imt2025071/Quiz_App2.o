# Quizora

An Android application for taking quizzes, with teacher and student modules.

## Features
- **Teacher View**: Setup quiz properties (Subject, Number of questions, Timer).
- **Student View**: Coming soon.

## How to open this in Android Studio

1. Open Android Studio.
2. Select **Open** an existing Android Studio project.
3. Choose the directory where you cloned this repository.
4. Android Studio will automatically configure the project and download requested Gradle wrappers/components if needed.
